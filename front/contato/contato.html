ABOUT US

Welcome to Freese Store. Discover our journey and the tech behind this project.

THE PROJECT

Developed by Lucas Freese, this project was created to help small stores that cannot afford expensive systems. It includes a complete online store for customers and an Admin Panel to easily manage products, sales, and stock.

DEVELOPMENT

Created with a focus on modern UI/UX design, fast performance, and a secure relational database to provide the best experience for both shop owners and clients.

TECHNOLOGIES USED

Here is the tech stack used to build this amazing e-commerce:

  • HTML5 & CSS3: Custom and responsive layout.
  • JavaScript (Vanilla): Dynamic front-end interactions.
  • Node.js & Express: Secure backend API and routing.
  • Supabase (PostgreSQL): Relational database management.
  • Render & GitHub: Cloud hosting and version control.
